# DSG Secure Deploy Gate Pro

This document defines the commercial packaging around the free GitHub Action.

## Free layer

The public GitHub Action provides:

- readiness endpoint check
- optional JSON ok check
- optional protected route check
- GO / NO-GO output
- SHA256 evidence hash
- GitHub Actions summary

## Pro layer

DSG Secure Deploy Gate Pro can add hosted release-governance capabilities:

- evidence ledger for every release
- deployment history dashboard
- monthly readiness reports
- Vercel deployment verification
- Supabase readiness and table reachability checks
- Slack, email, and webhook alerts
- custom policy rules
- team approval workflow
- exportable audit evidence bundle
