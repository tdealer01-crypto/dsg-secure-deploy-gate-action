# Pricing

## Free Action

The GitHub Action is free and open source.

Included:

- readiness endpoint check
- optional `ok: true` JSON check
- protected route check
- GO / NO-GO output
- deterministic evidence JSON
- evidence/proof/chain hashes
- optional PR comment

## Planned paid control plane

Paid hosted control-plane features are planned and should not be described as live until billing and access flows are actually available.

Planned features may include:

- proof history dashboard
- multi-repo evidence timeline
- team approvals
- Slack/webhook alerts
- audit export
- policy templates
- longer evidence retention

## Claim boundary

This Action can support compliance evidence workflows, but it is not itself PDPA, ISO 27001, SOC 2, WORM, or legal certification.
